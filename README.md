- **Database**: MongoDB integration via Beanie engine for user management and content storage.

## üõ†Ô∏è Getting Started

Follow these instructions to get the platform running on your local machine.

### Prerequisites
- Node.js (v18 or higher)
- Python 3.10+
- PostgreSQL database

### 1. Clone the Repository
```bash
git clone https://github.com/ANKUSH-alt/blog.git
cd blog
```

### 2. Setup the Backend
Navigate to the backend directory and set up your Python environment:
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
pip install -r requirements.txt
```

**Configure Backend Environment Variables:**
Create a `.env` file in the `backend/` directory:
```env
# Database
MONGODB_URL=mongodb://localhost:27017
MONGODB_DB_NAME=aieracademy_db

# AI and External APIs
GEMINI_API_KEY=your_gemini_or_openrouter_api_key
UNSPLASH_SECRET_KEY=your_unsplash_secret_key
# Other necessary keys like JWT_SECRET...
```

**Run the API Server:**
```bash
uvicorn app.main:app --reload
```
The backend API will run natively on `http://localhost:8000`.

### 3. Setup the Frontend
Open a new terminal window, navigate to the frontend directory:
```bash
cd frontend
npm install
```

**Configure Frontend Environment Variables:**
Create a `.env.local` file in the `frontend/` directory:
```env
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_UNSPLASH_ACCESS_KEY=your_unsplash_public_access_key
```

**Run the Development Server:**
```bash
npm run dev
```
The frontend application will run natively on `http://localhost:3000`.

## üîí Security & Best Practices

- **API Keys**: All sensitive API keys are stored securely in `.env` / `.env.local` files, which are strictly ignored from version control via `.gitignore`.
- **Image Fallbacks**: Robust image intercepting logic in the frontend ensures that Unsplash rate limits never break the user interface. Fallback SVGs keep the site looking perfect.
- **Sanitized AI Inputs**: External URLs generated by AI are strictly sanitized and URL-decoded in the frontend before fetching external APIs.
